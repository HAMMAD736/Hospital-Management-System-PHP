# 🏥 Hospital Management System (HMS)

A web-based application to manage hospital operations, appointments, and staff. Built using **Core PHP, MySQL, and Bootstrap 5**.

## 🚀 Features
- **Admin Panel:** Manage Doctors & Patients (CRUD Operations).
- **Doctor Portal:** View Appointments & Approve Status.
- **Patient Portal:** Book Appointments Online.
- **Live Status:** Real-time Pending/Approved status tracking.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Backend:** PHP (Native)
- **Database:** MySQL

## ⚙️ How to Run
1. Import the `hms.sql` file into phpMyAdmin.
2. Update `db.php` with your database credentials.
3. Open `index.php` in your browser.
